在激光光学、天文观测和半导体光刻等高精度光学系统中，抗反射涂层作为核心组件，直接影响透光效率、成像质量与系统稳定性。传统设计依赖经验公式或静态优化策略，难以同时实现超低反射率、高波前质量与工业化生产可行性。本文提出一种多层薄膜与双尺度纳米结构混合的数学优化模型，通过L-BFGS-B算法在六维参数空间中实现反射率、波前均方根误差和表面均方根误差的多目标协同最小化。模型基于传输矩阵法计算多层V-H-V-H-V结构的反射率，融入纳米表面高度函数的梯度修正、亚波长调制和径向优化以获得总反射率；波前质量采用Zernike多项式校正低阶像差；表面粗糙度通过曲率代理量化；优化目标函数采用加权和形式，优先强化低反射性能的同时平衡波前质量与制造约束。数值结果表明，平均反射率降至0.1322\%，中心反射率0.1495\%，均匀性10.7408\%；校正后波前均方根误差为11.5528毫波长；表面高度标准差7.6999nm；光谱半高全宽120 nm。该模型提升了设计精度与计算效率，适配UV纳米压印和电子束蒸镀等工艺，推动高效能耗光学器件开发，在高功率激光器件、半导体光刻系统及航天激光雷达等领域具有显著理论与工程价值。
